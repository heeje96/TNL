## Q4 리스트 총합 구하기

## Q5 피보나치 함수

첫 번째 항의 값이 0이고 두 번째 항의 값이 1일 때, 이후에 이어지는 항은 이전의 두 항을 더한 값으로 이루어지는 수열을 피보나치 수열이라고 한다.

```python
0, 1, 1, 2, 3, 5, 8, 13, ...
```

입력을 정수 n으로 받았을 때, n 이하까지의 피보나치 수열을 출력하는 함수를 작성해 보자.

### A5

```python
a = list()
a += [0, 1]
n = int(input())

for i in range(2, n):
    a += [a[i-1] + a[i-2]]

if n != 0:
    print(a)
else:
    print(0)
```
