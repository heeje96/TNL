# 개요
Web RTC and Websockets을 활용한 화상미팅

# 설치된 라이브러리 및 설치명령어
```
npm i nodemon -D
npm i @babel/core @babel/cli @babel/node -D
npm i @babel/preset-env -D
npm i express
npm i pug
npm i socket.io
npm i -g localtunnel
```
### 명령어
```
//dev로 구동시키기
npm run dev

//로컬터널 서버열기
lt --port 3000
```
